Contexto

Time de TI recebia chamados por WhatsApp pessoal, e-mail e formulário sem centralização. Alta taxa de tickets esquecidos, sem priorização por urgência e com tempo de resposta inconsistente entre os canais.

Escopo da implementação

Desenvolvimento de agente com Claude API que processa mensagens recebidas, identifica categoria e nível de urgência, gera resposta automatizada para casos recorrentes e cria tickets estruturados no GLPI via API para os casos que demandam atenção humana.
